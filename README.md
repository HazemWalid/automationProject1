#  E-commerce Test Automation Framework

##  Overview

Automated testing framework for an e-commerce web application using Selenium WebDriver, TestNG, and Cucumber (BDD).

##  Tech Stack

* Java (JDK 17)
* Selenium WebDriver
* TestNG
* Cucumber (BDD)
* Maven
* IntelliJ IDEA

##  Features Covered

* User Login
* Search Functionality
* Product Wishlist
* Currency Selection
* Home Sliders
* Hover Categories
* Social Media Links

##  Notes

* Registration test excluded due to CAPTCHA limitation.
* Explicit waits implemented to handle dynamic elements.

##  How to Run

```bash
mvn test
```

##  Reporting

* TestNG Reports
* Cucumber HTML Reports
